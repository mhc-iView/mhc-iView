## 基于iView的Vue.js组件库

[在线DEMO](https://luolr.github.io/mhc-iView-doc)

### 编译运行

``` bash
# 克隆项目
git clone https://github.com/mhc-iView/mhc-iView.git

# 安装依赖
npm install

# 本地运行
npm run start
```
